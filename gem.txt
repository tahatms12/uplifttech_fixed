1  Place **\_redirects** and **\_headers** inside /public (Vite copies everything under /public to the output). Delete any earlier copies in src/ or root. Redeploy.

2  Alternate: remove both files and put a single **netlify.toml** in project root:

```toml
[[redirects]]
  from = "/sw.js"
  to   = "/sw.js"
  status = 200

[[redirects]]
  from = "/service-worker.js"
  to   = "/service-worker.js"
  status = 200

[[headers]]
  for = "/*"
  [headers.values]
  Content-Security-Policy = """
default-src 'self';
script-src 'self' 'unsafe-inline' 'unsafe-eval' https://www.googletagmanager.com https://www.google-analytics.com https://connect.facebook.net https://static.ads-twitter.com https://snap.licdn.com https://q.quora.com https://bat.bing.com https://td.doubleclick.net https://sc-static.net https://cdn.pinterest.com;
connect-src 'self' https://www.google-analytics.com https://region1.google-analytics.com https://stats.g.doubleclick.net https://bat.bing.com https://tr.snapchat.com https://analytics.twitter.com https://q.quora.com;
img-src 'self' data: blob: https://www.google-analytics.com https://www.googletagmanager.com https://connect.facebook.net https://static.ads-twitter.com https://snap.licdn.com https://q.quora.com https://tr.snapchat.com https://cdn.pinterest.com https://stats.g.doubleclick.net;
frame-src 'self' https://forms.zohopublic.com https://www.googletagmanager.com https://td.doubleclick.net;
font-src 'self' https://fonts.gstatic.com;
worker-src 'self';
base-uri 'self';
object-src 'none';
form-action 'self'
"""
```

3  Delete vite-plugin-pwa from vite.config.js and package.json. Re-run `npm i`, `npm run build`, `netlify deploy`. Navigation-preload warnings disappear because no service worker registers. Re-enable only after sw is fixed.

4  To keep the service worker and silence the preload warnings, add to sw\.js top level:

```js
self.addEventListener('activate', e =>
  e.waitUntil(self.registration.navigationPreload.disable())
);
```

5  Font errors: replace the two .woff2 files with valid copies from Google Fonts. Confirm size > 0 B. Commit to /public/fonts. In CSS keep:

```css
@font-face{
  font-family:'Poppins';font-style:normal;font-weight:400;
  src:url('/fonts/poppins-latin-400-normal.woff2') format('woff2');
}
@font-face{
  font-family:'Poppins';font-style:normal;font-weight:600;
  src:url('/fonts/poppins-latin-600-normal.woff2') format('woff2');
}
```

6  Rebuild, redeploy, hard-refresh (Ctrl + Shift + R). Console must show: no CSP blocks, no font decode, no “Unexpected token '<'”, no navigation-preload cancellations.
